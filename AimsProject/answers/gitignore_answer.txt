1. Thêm các đường dẫn trên vào .gitignore vì:
Các file/thư mục này được sinh ra tự động bởi Eclipse IDE.
Chúng là file cấu hình cá nhân của IDE (không liên quan đến source code), file build tạm thời (bin chứa .class).
→ Không cần thiết đẩy lên Git và có thể gây xung đột giữa nhiều người dùng IDE khác nhau
→ Làm repository nặng và chứa dữ liệu rác

2. Nếu dùng IntelliJ IDEA thì thêm:
.idea/
*.iml
out/
còn nếu dùng VSCode thì thêm:
.vscode/

3. Nếu đã lỡ commit các file rồi và muốn Git ngừng theo dõi mà không xóa khỏi máy, ta dùng lệnh:
git rm -r --cached .
git add .
git commit -m "Stop tracking IDE and build files"
Hoặc nếu chỉ muốn bỏ theo dõi một số file:
git rm -r --cached .idea *.iml out
git commit -m "Remove IDE and build files from tracking"
